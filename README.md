ATM Transaction Simulation

This project is a console-based ATM Transaction Simulation developed using C programming language.


Features

      1.Create new bank account

      2.PIN-based authentication
  
      3.Deposit and withdraw money
  
      4.Balance enquiry
  
      5.Mini statement
  
      6.Transaction history with date and time
  
  
Technologies Used

       1.C Programming
  
       2.Structures
  
       3.File Handling
  
       4.Time library
  
  
How to Run

      Compile the program using a C compiler
  
      Run the executable file
  
  Follow the menu-driven instructions
